
# FrontendDevReactjs - Ezra Valenne Tofa

## Project Info

- Frontend: ReactJS (v19.x), Vite (v8.x)
- Backend: Node.js (v18+), Express (v4.x)

#### Frontend
```bash
npm install
npm run dev
```

#### Backend
```bash
cd server
npm install
npm run dev
```

- Project dihosting di: [Railway](https://railway.com)
